COVID-19 Global Effect Analysis

This project explores the impact of the COVID-19 pandemic across different countries using real-world datasets from Johns Hopkins and World Bank. It covers case trends, mortality rates, recovery, healthcare capacity, and economic indicators.

  • -Used Python (Pandas, Matplotlib, Plotly) for data processing and visualization
  • -Created country-level dashboards and global heatmaps
  • -Analyzed correlation between healthcare infrastructure and fatality rate
